### System Requirements

* Node.js 19.4.0 +
* npm version 9.2.0 +

### Installation:

```sh
$ make install
```

## Games:

* ***brain-even***
  - Try to figure out number is even
* ***brain-calc***
  - Try to solve the expression
* ***brain-gcd***
  - Try to find the greatest common divisor of two numbers
* ***brain-progression***
  - Try to find missing number
* ***brain-prime***
  - Try to figure out number is prime
